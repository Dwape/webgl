# WebGL
Moving a 3D object

## Controls

`w` to go forward<br/>
`a` to move left<br/>
`s` to go backwards<br/>
`d` to move right<br/>
`Space` to go up<br/>
`e` to rotate counter-clockwise<br/>
`q` to rotate clockwise<br/>
